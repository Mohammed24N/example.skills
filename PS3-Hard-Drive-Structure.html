The Structure of a PS3 Hard Drive

All of the HDD models used in the Sony Playstation 3 are 2.5" in size and 9 millimeters height max. These Hard Drives can be taken out and upgraded up to a maximum limit of 2TB of storage. There are currently 4 brands of PS3 Hard Drives: Fujitsu, Hitachi, Seagate, and Toshiba.

PS3 File Systems and Data Storage

FAT32 is the 32-bit version of the file allocation table (FAT) File System. A File System specifies the protocol for storing and organizing the data that is found on a Hard Drive with filenames and certian permissions.

The PS3 Hard Drive is formatted with a modified UFS2 file system. It also uses heavy encryption to protect its data, whcih makes it difficult to extract or modify files outside of the PS3 File System. The PS3 supports external Hard Drives but they have to be formatted in FAT32.
